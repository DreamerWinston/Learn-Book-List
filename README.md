## 前端全栈书籍
因为当当网书香节和京东读书月,我入手了我的第一批实体书,当阅读完以后,我会挨个发表读书笔记~希望各位继续关注

本人挑选实体书两个准则:
1.**近一年出版的书,网络上没有电子版.**
2.**有电子版的书,但是需要反复研读的,例如CSS世界.**
3.**工具书,需要随时可以进行翻看的.**

网络信息化时代,实体书存在的意义,便是阅读之后知识的沉淀,其吸收知识的效率,要比看视频和直播快.
